LifeOS

Your AI-First Linux Distribution

🚀 Quick Start

Your system is ready! Here are the essential commands:

  • life status - Check system status
  • life ai chat - Start AI assistant
  • life update - Update system
  • life config show - View configuration

🤖 AI Assistant

LifeOS includes a local AI assistant powered by Ollama. All processing happens on your machine - no data leaves your computer.

  • Chat with AI locally
  • Code assistance
  • Document analysis
  • Privacy-first design

🔄 Atomic Updates

LifeOS uses bootc for reliable, atomic system updates. If something goes wrong, you can always roll back.

  • Safe, atomic updates
  • Automatic rollback on failure
  • Multiple boot entries
  • No "dependency hell"
