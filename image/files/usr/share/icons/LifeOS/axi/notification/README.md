# Axi Notification Icons - 22x22

This directory contains minimal notification icons of Axi's head only.

## Purpose

These icons are optimized for system notifications where space is limited.
They show only Axi's head with expression matching the system state.

## Generation

```bash
# Generate from SVG head crops
make axi-pngs
```

## Usage

- Desktop notifications
- System tray indicators
- Status bar icons
- Compact status displays
