# Axi PNG Assets - 512x512

This directory contains high-resolution PNG exports of Axi the Axolotl.

## Generation

Run from the project root:

```bash
make axi-pngs
```

This requires either:
- **Inkscape** (`dnf install inkscape`)
- **librsvg2-tools** (`dnf install librsvg2-tools`)

## Files

| File | Description |
|------|-------------|
| `axi-healthy.png` | Axi base state - smiling, relaxed gills |
| `axi-updating.png` | Axi with construction helmet - updates in progress |
| `axi-rollback.png` | Axi regenerating - green glow effect |
| `axi-autonomy.png` | Axi with intelligence glasses - Autonomy mode |
| `axi-focus.png` | Axi with headphones - Focus/Flow mode |
| `axi-meeting.png` | Axi with tie - Meeting mode |
| `axi-night.png` | Axi in pajamas - Night mode |
| `axi-error.png` | Axi worried - Error state |
| `axi-offline.png` | Axi sleeping - Offline state |
